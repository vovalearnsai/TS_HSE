# HSE-Time-Series-2025

Содержание:
- Class_1 — Введение, EDA, предобработка, метрики, валидация.
- Class_2 (Практика) — EDA, метрики, STL-разложение.
- Class_3 — Модели экспоненциального сглаживания. ETS-модели.
- Class_4 — Стационарность. ACF, PACF. SARIMA(X)-модели.
- Class_5 — Автоподбор спецификаций, анализ прогнозов и остатков, многомерные ряды.
- Class_6 (Практика) — Перевод задачи прогнозирования в задачу машинного обучения. Feature eng. Глобальные модели.
- Class_7 (Практика) — Многомерное и многошаговое прогнозирование.
- Class_8 — Нейронные сети для прогнозирования временных рядов. RNN, LSTM. Специализированные Dlinear, CycleNet.
- Class_9 — Нейронные сети для прогнозирования временных рядов. CNN. Специализированные DeepAR, WaveNet, TCN, LSTNet, xLSTM.
- Class_10 — Нейронные сети для прогнозирования временных рядов. Transformer. Специализированные TFT, Informer, Autoformer, FEDformer, PatchTST, iTransformer. Фундаментальные модели LLMTime, Chronos, TimesFM, Moirai, Moirai-MOE.
- Class_11 (Практика) — Нейронные сети для прогнозирования временных рядов. Практические советы по обучению и отладке. Сравнение моделей.
